## 📌 Overview

This project showcases a **complete end-to-end Data Analytics workflow** — from raw dataset extraction to advanced SQL-based business insights.

The dataset was sourced from **Kaggle**, and the project demonstrates **data cleaning, transformation, ETL pipeline creation, and business analysis** using **Python and SQL Server**.
